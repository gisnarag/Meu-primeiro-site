Exemplo de Código

Como criar um documento HTML mínimo

Para começar o HTML use a tag <!DOCTYPE pois você está definindo que esse é tipo o documento que você vai criar. Em seguida, há uma 'root' (tag principal), que é a tag <html>, dentro dela há dois filhos importantes <head> e <body>. A <head> é onde você configura a sua página, e o <body> é onde você coloca o conteúdo que deseja exibir ao usuário, e lembre-se de fechar as tags corretamente.

Exemplo:


        <!DOCTYPE>
        <html>
        <head>     </head>
        <body>     </body>
        </html>
